# Smart Report Extractor — Frontend

React + TypeScript UI for uploading PDFs and viewing structured extraction results.

## Stack

- React 18 + TypeScript
- Vite
- Axios

## Setup

```bash
npm install
npm run dev
```

App runs on `http://localhost:5173`. Backend must be running on `http://localhost:3000`.

## Usage

1. Click **Choose File** and select a PDF (invoice, bank statement, or any document)
2. Click **Upload PDF**
3. View the extracted fields and AI summary in the results panel
